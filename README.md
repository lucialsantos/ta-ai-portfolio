# Lucía Santos — TA Ops × AI Portfolio

A portfolio that is also a bootcamp. 14 weeks. Built in public.

Transitioning from Tech Talent Acquisition to TA Operations Partner (AI & Product) at Factorial. Every project here is real — specs an engineer can build from, tools the TA team can use, automations that remove manual work.

## Update your progress

Edit `bootcamp-progress.json` at the project root:
- Change `status` → `"in_progress"` or `"completed"`
- Add `startedAt` / `completedAt` dates
- Add `url` when a project is deployed

Then `git add . && git commit -m "update phase X progress" && git push` — the site rebuilds automatically on Vercel.

## The daily git ritual

```bash
git add .
git commit -m "what I did today"
git push
```

That's it. Vercel does the rest.

## Local dev

```bash
npm run dev
```

Open http://localhost:3000
